Control Flow and Loops


Some of the things Javascript uses to create these fun "toys" for us are 'Control Flow' and 'Loops'.

A control flow is kind of like a shopping list, where you carefully collect each item checking them off the list as you go.
but a loop is more like having to wake up and go to school or work every day. Just like a control flow, Loops also work from a list but a loop will continue to go through the list over and over again until something on the list changes. Like going to work everyday until a public holiday comes around to give you some time off.

Interacting with the DOM


Lets say i wanted to make changes to an HTML element like all the different paragraphs on this page, how would i do that?
while first you give the paragraph element or p in HTML, a class name like idk pineapples. What this does is it links all of the paragraphs with the same class name togther in a little chain so that we can change all of them together.

now lets say i wanted to create a loop while first i would need that list right? and that is where the HTML DOM comes into play by allowing us to create an array (i'll be talking more about this later so don't worry about it for now). and the way we access this array is with a little command called document.getElementsByClass(“name-of-class”) which collects all the elements with our class name aka 'pineapples' and gives it back to us as an array or our list.

Differences between Array's and Object's


Now you must be thinking "that's really cool but what if instead of a list i just want to interact with a picture or video?" while i'm glad you asked reader, that's where objects come into play but before i talk about that lets finish talking about array's like i promised.

An Array is esentially sets of data between [square, brackets] separated by commas. which is a great way to store collections and lists that can be accessed by their position within the array. To access the array we use zero-based indexing, which means the first item in an array is 0, the second item is 1, the third is 2 etc.

Where as an Object is used to represent a “thing” in your code and are expressed by { curly brackets}. That could be a person, a car, a building, a book, a character in a game — basically anything that is made up or can be defined by a set of characteristics. In objects, these characteristics are called properties that consist of a key and a value.

What are Functions and there uses


"so that's how you access all the information but how do you get it to do something?", well my dear reader that is where something called 'Functions' come into play.
A function as the name implies is what allows our web page to perform a task such as telling a button to let us click it and make something happen, or in other terms 'return a value'.

"but why do we use functions?", because my sweet reader it allows us to perform the same task without having to constantly rewrite code, it also allows us to use one code to create many different out comes.

like if i press a button once it turns blue, if i press the same button again it turns red but if i double click the button it becomes purple. These are all made possible because of the function code, sounds pretty useful am i right?.
